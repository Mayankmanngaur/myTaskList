Task Manager - Web Application

This is a simple and responsive **Task Manager** built using **HTML**, **CSS**, and **Vanilla JavaScript**.

Features

- Add new tasks easily
- Mark tasks as completed
- Delete tasks
- Filter tasks: All, Done, Pending
- Tasks stored in browser LocalStorage
- Sort tasks by newest first
- Fully mobile responsive design

Live Demo

[Click here to view the live project](http://my-task-list-mocha.vercel.app)

How to Use

1. Enter your task in the input box.
2. Click 'Add Task' to add it to the list.
3. Click on a task to mark it as completed.
4. Click 'X' button to delete a task.
5. Use the filters (All / Done / Pending) to view specific tasks.

Technology Stack

- HTML5
- CSS3 (Responsive Design)
- Vanilla JavaScript
- LocalStorage (for saving tasks)
